# OCR Extractor

โปรแกรม OCR เดสก์ท็อปสำหรับอ่านข้อความไทยและอังกฤษจากภาพจำนวนมาก พร้อม preview, ค้นหา, ส่งออก และรายงานการทำงาน

## เริ่มใช้งาน

1. ติดตั้ง Python 3.10 หรือ 3.11 (64-bit) และอัปเดต pip
2. เปิด Terminal ในโฟลเดอร์นี้ แล้วรัน `python -m pip install -r requirements.txt`
3. รัน `python main.py`

ครั้งแรก PaddleOCR จะดาวน์โหลดโมเดลภาษาไทย จึงต้องเชื่อมต่ออินเทอร์เน็ตเพียงครั้งเดียว

## การใช้งาน

- เลือกหลายไฟล์ หรือเลือกโฟลเดอร์ (รวมไฟล์ในโฟลเดอร์ย่อย)
- กด **เริ่ม OCR** แล้วพัก/ทำต่อ/หยุดได้
- คลิกภาพทางซ้ายเพื่อดูผล; คลิกบรรทัดข้อความเพื่อเน้นกรอบบนภาพ
- ปุ่ม TXT/DOCX/XLSX/CSV/JSON/PDF ส่งออกผลรวมไปยัง `output/`; และสร้างไฟล์ `.txt` แยกต่อภาพอัตโนมัติ
- `output/history.json` เก็บประวัติ 100 งานล่าสุด และ `output/errors.json` เก็บข้อผิดพลาดของรอบล่าสุด

## หมายเหตุ

- PDF input ยังไม่เปิดใช้ตามขอบเขตที่ระบุว่าเพิ่มภายหลัง
- GPU ใช้ได้เมื่อเปลี่ยนไปติดตั้ง PaddlePaddle รุ่น CUDA ที่ตรงกับเครื่อง แล้วปรับ `use_gpu=True` ใน `ui/main_window.py`
- การส่งออก PDF ด้วย ReportLab อาจต้องติดตั้ง/ลงทะเบียนฟอนต์ไทยเพิ่มเติม หากต้องการให้ตัวอักษรไทยฝังใน PDF ทุกเครื่อง
